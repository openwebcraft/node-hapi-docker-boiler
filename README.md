# node-hapi-docker-boiler

This is a boilerplate for developing a small hapi server inspired by [openwebcraft/node-js-starter](https://github.com/openwebcraft/nodejs-docker-starter)

## dependencies

* Hapi
* mongoose
* bluebird
* boom
* hapi-auth-jwt2
* joi
* jsonwebtoken
* moment
* require-all

## devDependencies

* code
* lab
* nock
* pow-mongodb-fixtures
